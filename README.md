# Groundwater Intelligence Platform

Aqua-Assess is a web-based data visualization and analysis platform designed to help users analyze, convert, and report groundwater pollution data efficiently. The platform focuses on simplifying raw dataset handling and enabling insightful visual exploration through interactive charts and reports.

<img width="1600" height="786" alt="image" src="https://github.com/user-attachments/assets/503068e7-35de-4328-8be5-a41b4629335d" />


## Features

- Interactive Data Visualization
- CSV Data Processing
- Export reports to PDF for easy sharing and documentation
- User-Friendly UI

## Project Structure


├── components/      # Reusable UI and visualization components
├── services/        # Data handling and processing logic
├── utils/           # CSV parsing and unit conversion utilities
├── App.tsx          # Main application component
├── index.tsx        # Application entry point
├── constants.ts     # Global constants
├── types.ts         # TypeScript type definitions
├── test_units.csv   # Sample dataset for testing
├── vite.config.ts   # Vite configuration
└── package.json     # Project dependencies and scripts

## Setup and Installation
1. Clone the repository
   ```
   git clone https://github.com/keshavguptase24b02024-eng/final_repo.git
   cd final_repo
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Run the development server
   ```
   npm run dev
   ```
